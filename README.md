## Features

### Placement Prediction

* Predict placement readiness based on academic and skill-related factors.
* Generate personalized placement insights.
* Store and review prediction history.

### ATS Resume Analysis

* Upload resumes in PDF format.
* Receive ATS compatibility scores.
* Extract and analyze skills.
* Identify resume improvement opportunities.

### Skill Analytics

* Visualize skill strengths and weaknesses.
* Compare skill levels against industry expectations.
* Discover areas for improvement.

### Internship Discovery

* Search internship opportunities.
* Explore opportunities based on skills and interests.
* Access internship details from integrated sources.

### User Profile Management

* Secure authentication and authorization.
* Degree and specialization management.
* Personalized user dashboard.

### History Tracking

* Resume Analysis History.
* Placement Prediction History.
* Progress monitoring and tracking.
