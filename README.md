# SkillAI - AI-Powered Placement Intelligence Platform

## Overview

SkillAI is an AI-powered placement intelligence platform designed to help students evaluate their placement readiness, analyze resumes using ATS-style scoring, identify skill gaps, and discover relevant internship opportunities.

The platform combines Machine Learning, Resume Analysis, Data Analytics, and Placement Prediction into a single user-friendly web application.

## Installation

### Clone Repository

```bash
git clone https://github.com/Mugundhan-02/SkillAI.git
cd SkillAI
```

### Frontend Setup

```bash
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

Frontend:

```text
http://localhost:5173
```

Backend:

```text
http://127.0.0.1:8000
```

API Documentation:

```text
http://127.0.0.1:8000/docs
```

## Key Functionalities

### Placement Prediction

Students can enter their academic and skill-related information to receive a placement probability score generated by a machine learning model.

### Resume ATS Analysis

Users can upload resumes and receive ATS-style scoring along with skill extraction and improvement recommendations.

### Internship Discovery

Students can search and explore internship opportunities relevant to their academic background and interests.

### Skill Analytics

The platform analyzes student profiles and highlights areas that require improvement for better placement opportunities.
